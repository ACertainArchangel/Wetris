# Welcome to the hardware log!

This is a place for us to put what pins and other hardware we want so we aren't using each other's stuff.

| Reserved Component | Name of User | Purpose |
|-------------------|--------------|---------|
| OC1A(pin11)+ Timer1 | Gabe | Square wave output for music |
| OC3A(pin5) + Timer3 | Gabe | Square wave output for music |
| OC4A(pin6) + Timer4 | Gabe | Square wave output for music |
| PCINT7 (pin13) | Roman+Jorge+Gabe | Left Movement |
| PCINT6 (pin12) | Roman+Jorge+Gabe | Right Movement |
| PCINT4 (pin10) | Roman+Jorge+Gabe | Rotation Movement |
| Pins for LCD (pins 7-9) | Roman | LCD connections |
| Pins for LCD(Dig. Pins50-52) | Roman | LCD connections |
| Pin for LCD LED (pin2) | Roman | Backlight control |
| FWD Pin (22) | Son+Gabe+Jorge | Drive actuator forward |
| REV Pin (23) | Son+Gabe+Jorge | Drive actuator reverse |
| A0 (pin A0)  | Son            | Potentiometer for game speed adjustment|
